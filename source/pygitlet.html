pygitlet.commands.add(repo: Repository, file_path: Path) → None¶

Stages a file. Overwrites existing staged files if the same named file exists and differs. If identical, the file is unstaged. If previously staged for removal, undoes this.

Parameters:

  • repo – PyGitlet repository.

  • file_path – Relative path to file to be staged.

Raises:

PyGitletException – If the requested file doesn’t exist.
